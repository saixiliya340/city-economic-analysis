# 中国城市薪资与生活成本分析

## 项目简介
使用 Python 对 23 个中国城市的平均薪资、房租、生活开销等数据进行多维度分析，并生成可视化图表。

## 使用工具
- Python (Pandas + Matplotlib)
- 数据来源：综合公开数据

## 分析内容
1. 各城市平均薪资排行
2. 房租占工资比例（租房压力对比）
3. 不同线城市薪资对比
4. 每月可支配收入（结余）对比
5. 薪资 vs 房租散点图

## 如何运行
`ash
pip install pandas matplotlib
python analysis.py
`
